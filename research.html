Project Mission

To build an agentic Retrieval-Augmented Generation (RAG) system that transforms our static library of 2000+ research articles into a dynamic, explorable knowledge base for organizing research, analyzing methodologies, and simulating experiments.

Project Roadmap

This is our four-phase plan to bring the agentic RAG system to life. Each phase builds upon the last, moving from foundational data work to advanced simulation. Click on any phase to explore its objectives and desired outcomes.

1

Phase 1: Foundation

Ingestion & Indexing

Key Objectives:
  • Automate ingestion of 2000+ articles (PDF, TXT, HTML).
  • Implement hierarchical chunking for semantic context.
  • Enrich all chunks with metadata (title, author, year, category).
  • Set up and populate a vector database for semantic search.
Desired Outcome:

A fully indexed, searchable knowledge base ready for the RAG system.

2

Phase 2: The Agentic Core

Reasoning & Retrieval

Key Objectives:
  • Develop a primary query agent for intelligent request handling.
  • Implement a hybrid search mechanism (semantic + keyword).
  • Create a context validation agent to grade retrieval relevance.
  • Enable iterative query refinement for failed searches.
Desired Outcome:

A conversational RAG system that can accurately answer complex questions about the corpus.

3

Phase 3: Relationship Analysis

Methodology Mapping

Key Objectives:
  • Build a pipeline for Named-Entity Recognition (NER) of methodologies.
  • Implement relationship extraction to understand how methodologies connect.
  • Generate and populate a knowledge graph with extracted relationships.
  • Create an interactive visualization to explore the knowledge graph.
Desired Outcome:

An interactive knowledge graph revealing the hidden network of methodologies in our research.

4

Phase 4: Experimentation

Hypothesis & Simulation

Key Objectives:
  • Create a hypothesis generation agent to identify research gaps.
  • Develop a simulation design agent to synthesize experimental plans.
  • Integrate with external simulation frameworks by generating parameter outputs.
Desired Outcome:

A co-scientist agent capable of proposing and designing novel, testable experiments.
